YMF825
======

## 概要

C# から [YMF825Board](https://yamaha-webmusic.github.io/ymf825board/intro/) に接続するためのライブラリラッパと各種ライブラリです。

## プロジェクト

### YMF825

C# 向けの YMF825 ドライバクラスです。FT232HL を介した SPI 接続や、YMF825 のための高レベル API を提供しています。 

## 動作確認環境

- Windows 10 64bit
- CBW-YMF825-BB rev0.1
